Setup Module

1.1 Company Setup

The Company Setup page allows you to add and manage your company's basic information, logo, and display settings. This setup ensures proper branding is reflected across login pages, reports, invoices, and bills.

System Path:
Menu Basic Settings Company Setup

Key Capabilities

Group Management

Selection of companies based on organizational groups.

Identity Fields

Full support for legal names, unique codes, and contacts.

Visibility Toggles

Control logo and name visibility across system modules.

Precision Layouts

Optimized logo dimensions for A4, A5, and POS outputs.

Extended Docs

Optional branch info display in reports and bills.

Transaction Logic

Toggle-based transaction schedule activation.

Setup Guide

01

Update Company Information

Complete the foundational identity fields to personalize your ERP environment.

Group: Dropdown selection
Company Name: Legal name
Short Name: Abbreviation
Code: Unique ID
Contacts: Phone & Email
Web: Official URL
Address: Use the rich-text editor for full location details.
02

Upload Branding Assets

Main Logo Green Button (Max 1MB)
Report Logo B&W for Invoices
Login Cover 1680 x 1121px Banner
03

Configure Display Settings

Feature Behavior & Logic
Logo Display Toggle for Login/Reports. Custom rem width.
Name Visibility Toggle visibility on Login, Reports, and Bills.
Branch Info Show specific outlet address in footers.
Schedule Enable time-restricted transaction entries.

Final Step: After completing or updating all fields, click the UPDATE button at the bottom of the page to save your changes.

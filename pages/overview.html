Purpose

The General Configuration section is used to set basic system rules and preferences. It helps control how the ERP system works across all departments, ensuring uniformity in reporting, security, and operational workflows.

1.2 Main Features

1. Basic Settings

Set up foundational elements like:

  • Company profile
  • Company logo, Cover Image, Size setup
  • Payment system setup

2. Territory Management

Organize your physical presence:

  • Manage geographic regions, zones, or areas
  • Outlet setup and status updates

3. Configuration

Global process setup:

  • Fiscal Year, Holiday, Dept., and Designations
  • Terms and Conditions & Notice management
  • Enable/disable features or customize behaviors

4. HR Management

Full employee lifecycle:

  • Employee Master Entry
  • Promotion, Transfer, Terminate, and Resign workflows

5. Role & User Management

Security and Identity:

  • Create users and assign specific roles
  • Manage login access and granular permissions

6. Designation & Role Map

Inherited access rights:

  • Link job designations with system roles
  • Ensures correct access based on job position automatically

7. Task Management

Collaboration tools:

  • Create, assign, and track tasks or to-do items
  • Internal collaboration and progress monitoring

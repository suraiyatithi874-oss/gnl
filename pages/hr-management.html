5.1 Employee Setup

Manage the complete employee lifecycle. This module provides tools for maintaining personnel records, tracking career progression, and handling organizational exits.

Purpose

To manage employee records, allowing users to add, edit, view, and delete employee profiles for HR or administrative tasks.

Navigation:
Menu HR Management Employee Setup

Main Features

Smart Filtering

Filter employees by branch, department, designation, and gender with precision.

Quick Search

Dedicated search bar to instantly locate staff members by name or unique ID code.

Employee Table

A structured, high-visibility list view displaying key details of all registered staff.

Action Icons

One-click access buttons to View profiles, Edit data, or Delete records efficiently.

New Entry

A prominent global button designed to initiate the full employee onboarding workflow.

How to Use

NEW ENTRY

Add a New Employee

  1. Click the + New Entry button at the top-right.
  2. Fill in the required details: Name, Code, Gender, Department, and Mobile.
  3. Click Save to commit the employee to the system list.
UPDATE DATA
  1. Locate staff in the table.
  2. Click the Edit icon ().
  3. Apply updates to the profile.
  4. Click Save.
VIEW DETAILS
  1. Locate the employee row.
  2. Click the View icon ().
  3. Review full profile in popup.
DELETE DATA
  1. Find the employee in list.
  2. Click Delete ().
  3. Confirm the system prompt.

4.2 Employee Promotion

Purpose

To record, track, and manage employee promotions within the organization, ensuring each promotion is documented with full details and status updates.

Navigation:
Menu HR Management Employee Promotion

Main Features

Advanced Search & Filters

Narrow down records by branch, department, designation, date range, status, or unique promotion code.

Promotion Logging

Add new records to capture career progression details accurately within the system.

Full Record Lifecycle

View detailed history, edit existing records, or delete incorrect entries as needed.

Navigation Tools

Integrated search bar and pagination for efficient record browsing across the database.

How to Use

NEW ENTRY

Log a Promotion

  1. Click the + New Entry button.
  2. Fill in the details:
    • Employee Basic Information
    • Current position (Designation/Dept)
    • Position after promotion
    • Promotion Date & Effective Date
    • Status (e.g., Approved, Pending)
  3. Click Save to finalize the record.
UPDATE DATA
  1. Find the specific record in the table.
  2. Click the Edit icon ().
  3. Update necessary info and click Save.
VIEW DETAILS
  1. Locate the promotion record.
  2. Click the View icon ().
  3. Review and check the promotion details.
DELETE DATA
  1. Find the record you want to remove.
  2. Click the Delete icon ().
  3. Confirm the permanent deletion.

4.3 Employee Transfer

Purpose

Purpose: To manage and track employee transfers between departments or branches, keeping the transfer records organized and accessible for HR administrative auditing.

Navigation:
Menu HR Management Employee Transfer

Main Features

Dynamic Filtering

Narrow down transfer records by branch, department, designation, date range, or current status.

Transfer Entry

Seamlessly add new transfer requests into the system with an intuitive data entry workflow.

Advanced Search

Instant search functionality to quickly locate specific transfer codes or employee names.

Record Management

Full administrative access to View details, Edit info, and Delete existing transfer records.

How to Use

NEW ENTRY

Log an Employee Transfer

  1. Click the + New Entry button at the top-right.
  2. Complete the required fields:
    • Transfer Code: Unique identifier for the transfer.
    • Employee Name: Select the employee being transferred.
    • Location Logic: Define both the 'From' and 'To' locations/branches.
    • Timelines: Input Application and Effective dates.
    • Status: Set the current state (e.g., Approved, Pending).
  3. Click Save to register the transfer.
UPDATE DATA
  1. Find the desired record in the table.
  2. Click the Edit icon ().
  3. Update information and click Save.
VIEW DETAILS
  1. Locate the specific transfer record.
  2. Click the View icon ().
  3. Review the complete transfer details.
DELETE DATA
  1. Find the entry you wish to remove.
  2. Click the Delete icon ().
  3. Confirm the deletion when prompted.

4.4 Employee Resign

Purpose

To record and manage employee resignation details, allowing HR/admin users to track, update, and review employee exit data systematically.

Navigation:
Menu HR Management Employee Resign

Main Features

Granular Filtering

Filter resignation records by branch, department, designation, resign code, date range, and status.

Comprehensive Management

View and manage all resignation records from a centralized, high-visibility table.

Detailed Entry

Capture specific exit data including resignation dates, application dates, and applicant names.

Actionable Records

Edit, view, or delete existing records using quick-access icons directly from the list.

How to Use

NEW ENTRY

Record a Resignation

  1. Click the + New Entry button.
  2. Enter Employee Info:

    Provide the Employee Name, Branch, Resign Date, Application Date, and the name of the person who applied for it.

  3. Set Status: Choose the appropriate status (e.g., Approved, Pending).
  4. Click Save to commit the entry.
UPDATE DATA
  1. Click the Edit icon () next to the entry.
  2. Modify the necessary fields and click Save.
VIEW DETAILS
  1. Click the View icon () to see the full resignation profile.
DELETE RECORD
  1. Click the Delete icon () for the specific entry.
  2. Confirm when prompted to remove the data.
UNDO DATA
  1. Click the Undo icon () to revert a resignation.
  2. Restores employee to Active Status.
